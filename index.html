Tải Lên Hình Ảnh Sinh Hoạt Mới

Hướng dẫn đồng bộ lưu dữ liệu vĩnh viễn lên GitHub của bạn

Do GitHub Pages là một trang web tĩnh nên mọi dữ liệu bạn vừa nhập/chỉnh sửa trên web hiện tại **chỉ đang được lưu tạm thời trên trình duyệt máy tính của bạn (LocalStorage)**.

Để phụ huynh trên mọi thiết bị khác cũng nhìn thấy những thay đổi này, hãy làm theo các bước sau:

  1. Bấm vào nút "Sao chép toàn bộ mã nguồn mới" phía dưới.
  2. Truy cập vào trang quản lý file index.html trong repository của bạn trên GitHub.
  3. Bấm nút sửa tệp (icon bút chì), xóa sạch toàn bộ mã cũ đi và dán đè mã mới vừa copy vào.
  4. Bấm Commit changes để hoàn thành. Sau 1 phút trang web của phụ huynh sẽ tự cập nhật!
