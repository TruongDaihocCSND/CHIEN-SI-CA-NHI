Hướng dẫn đồng bộ mã nguồn tĩnh lên GitHub của bạn

Nếu bạn không sử dụng Hệ thống đám mây trực tuyến, mọi dữ liệu bạn chỉnh sửa hiện tại sẽ được lưu tạm thời trên trình duyệt máy tính của bạn (LocalStorage).

Để lưu trữ cố định vào mã nguồn tĩnh của trang GitHub Pages:

  1. Bấm vào nút "Sao chép toàn bộ mã nguồn mới" phía dưới.
  2. Truy cập vào trang quản lý file index.html trong repository của bạn trên GitHub.
  3. Bấm nút sửa tệp (icon bút chì), xóa sạch toàn bộ mã cũ đi và dán đè mã mới vừa copy vào.
  4. Bấm Commit changes để hoàn thành.
